Задачи дашборда:
- Оценить общее финансовое состояние бизнеса. Быстрый доступ к показателям выручки и прибыли селлера.
- Выявить тренды и сезонность. Визуализировать, как меняются ключевые показатели во времени.
- Выделить наиболее важные товары/категории, которые приносят основную выручку и прибыль.
- Определить, кто из менеджеров приносит наибольшую прибыль, кто работает в убыток.
- Обеспечить контроль ключевой статьи расходов — логистики.


Метрики:
1. Ключевые показатели KPI.
2. Динамика продаж по месяцам. 
3. ABC-анализ по объему выручки, с детализацией по количеству заказов и среднему чеку.
4. Структура выручки по сегментам покупателей (Small Business, Consumer, Corporate, Home Office).
5. Перечень менеджеров по продажам с указанием их вклада в прибыль (или убыток).
6. Затраты на логистику.
