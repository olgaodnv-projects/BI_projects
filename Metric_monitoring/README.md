Дашборд предназначен для анализа продаж и мониторинга ключевых бизнес-показателей на маркетплейсе Wildberries.

Основные задачи:
- Отслеживание финансовых результатов за выбранный период
- Анализ эффективности продаж по товарам и регионам
- Мониторинг рентабельности бизнеса
- Выявление лидеров роста и падения выручки по товарам
- Контроль за распределением продаж по регионам

Ключевые метрики:
- Выручка 
- Количество заказов 
- Комиссия WB 
- Логистические расходы
- Валовая прибыль
- Рентабельность

Дополнительные элементы анализа:
- Динамика ROI
- Топ товаров по приросту и падению выручки
- Динамика выручки по месяцам
- Географическое распределение продаж по регионам России
- Статистика по возвратам
